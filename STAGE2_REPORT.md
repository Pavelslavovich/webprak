# Отчет по этапу 2

## Выполненные пункты задания

1. Разработка связывающего кода с помощью Hibernate.
2. Разработка классов хранимых объектов.
3. Разработка отображения БД на объекты в Hibernate.
4. Разработка служебных классов (DAO) с методами типовых запросов.
5. Разработка модульных тестов TestNG для созданных DAO-методов.
6. Дополнение Ant-сборки задачами компиляции и запуска тестов.

## Структура реализации

- Классы модели: src/main/java/ru/Pavelslavovich/webprak/model
- DAO-классы: src/main/java/ru/Pavelslavovich/webprak/dao
- Конфигурация Hibernate: src/main/resources/hibernate.cfg.xml
- Тесты TestNG: src/test/java/ru/Pavelslavovich/webprak/dao
- Тестовая конфигурация Hibernate: src/test/resources/hibernate-test.cfg.xml
- Сборка и зависимости: build.xml, ivy.xml, testng.xml

## Список нетривиальных запросов, покрытых тестами

- Поиск клиентов по услуге, интервалу времени и (опционально) по задействованному служащему.
- Поиск служащих по участию в договорах заданного клиента за период.
- Поиск услуг по части названия и диапазону базовой стоимости.
- Регистрация договора с прикреплением нескольких служащих и проверкой сохранения связей.
- Поиск договоров по статусу и номеру договора.

## Инструкции запуска

1. ant resolve
2. ant compile
3. ant compile-tests
4. ant test

Дополнительно для SQL-сценариев этапа 1:

- ant db.create
- ant db.init
- ant db.reset
